## 当前方向

当前较明确的前端概念方向是：

- `白底黑线` 的极简研究型界面
- 中央为 `蒲公英 / 原则花盘` 可视化
- 左右为有边界的轻量面板
- 下方为 `workflow`、运行记录与回写信息
- 整体气质偏 `diagrammatic interface`，而不是密集的 `dashboard`
