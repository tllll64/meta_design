# 任务清单

## 任务背景

你将以信息设计师的身份，为一场由高校设计学院与互联网公司设计团队联合举办的 AI 主题分享会设计宣传页面。请结合下方 `2` 个任务详细描述，完成对应的信息设计任务。

建议时间安排为：任务 1 约 `15` 分钟，任务 2 约 `10` 分钟。时间不做严格限制，你可以根据实际情况调整。

## 产出要求

1.请选择一种你习惯的输出形式：

- PPT (16:9)；
- 海报类视觉物料 (A4 竖版)；
- 社媒图文物料 (3:4)。

2.请按你平时最自然的方式完成任务，发挥独特的灵感与创意：

- 使用 `灵感网站`（如 Pinterest） 寻找排版、风格灵感；
- 使用 `设计工具`（如 Figma） 绘制草稿；
- 鼓励使用任何熟悉的 `AI 工具`（LLM  / AI Coding / Skills）产出最终效果。

3.在实验过程中，请大胆和我分享自己的想法（Speak Aloud）：

- 你拆解任务的思路；
- 你如何给 AI 指令；
- 你如何评价 AI 的输出。

<br />

任务更关注你在过程中的真实想法，不追求一定做出特别完整的最终成品。

# 任务详细描述

### 活动背景：

当设计师遇见 AI：从 Prompt 到 Workflow

这是一场由 `同济大学设计创意学院` 与 `小红书设计团队` 联合发起、由 `Future Workflow` 社群协办的主题分享会。活动聚焦 AI 如何真正进入设计工作流，而不只是停留在“写 prompt”，更希望讨论设计师如何在真实任务里与 AI 形成稳定、可复用、可控的协作方式。

### 活动调性：

- 清晰、专业、当代、带一点 `科技感` ;
- 不要太像商业发布会，更像有设计感的校企联合分享活动。

<br />

```text

核心亮点：

- 设计相关学生
```
